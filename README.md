# SchoolTask

Aplicacion web en Next.js + TypeScript + Tailwind CSS.

## Requisitos

- Node.js 20 o superior
- npm 10 o superior

## Ejecutar proyecto

1. Instalar dependencias:
   - `npm install`
2. Copiar el logo adjunto a:
   - `public/logo-schooltask.png`
3. Iniciar el proyecto:
   - `npm run dev`
